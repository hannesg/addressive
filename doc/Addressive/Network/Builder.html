Overview

A Builder is used to construct a network. It’s here so that a Addressive::Network can have only read methods, while a Builder does the heavy lifting. This class should not be generated directly, it’s created for you by #build and #initialize.

def node(name, &block)
  n = @network.node(name)
  if block
    if block.arity == 1
      yield @network.node_builder_class.new(@network, n)
    else
      @network.node_builder_class.new(@network, n).instance_eval(&block)
    end
  end
  return @network.node_builder_class.new(@network, n)
end
