Overview

A URISpecFactory contains all information necessary to create a URISpec. This class exists because it’s annoying to pass around default values by hand and a factory makes it possible to apply post-processing options.

def normalize( spec, defaults = self.all_defaults )
  if defaults.key? :app 
    spec.app = defaults[:app]
  end
  unless spec.template.absolute?
    if defaults[:prefix]
      spec.template = URITemplate.apply( defaults[:prefix] , :/ ,  spec.template)
    end
  end
  return spec
end
